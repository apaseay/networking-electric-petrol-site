Project: 'A4' created on 2026-02-21
Author: Ayobami Pase


GNS3 implementation of a hierarchical enterprise network architecture.
Core Layer 3 switches provide inter VLAN routing and VRRP based gateway redundancy.
Distribution and access switches implement VLAN segmentation, trunk links with native VLAN 999, port security, and STP root configuration.
Topology designed to demonstrate loop prevention, traffic isolation, and redundancy principles.
